# State Farm Insurance Companies

State Farm Insurance Companies is the collective name for the group of insurance subsidiaries operating under State Farm Mutual Automobile Insurance Company, the parent organization headquartered in Bloomington, Illinois. The group comprises fourteen property and casualty insurance companies and two life insurance companies. This multi-entity structure allows State Farm to manage its business across different US state regulatory environments. As a group, State Farm Insurance Companies is the largest property and casualty insurer in the United States, sharing the common digital infrastructure and developer platform operated at developer.statefarm.com.

**Website:** [https://www.statefarm.com](https://www.statefarm.com)  
**Developer Portal:** [https://developer.statefarm.com](https://developer.statefarm.com)

## APIs

### Partner Gateway API

The unified integration layer for all State Farm Insurance Companies subsidiaries, enabling external partners to access insurance products across the State Farm family of companies through a single API entry point.

- **Documentation:** [https://developer.statefarm.com](https://developer.statefarm.com)
- **FAQ:** [https://developer.statefarm.com/faq](https://developer.statefarm.com/faq)

### Renters Insurance API

Enables property management platforms, multifamily housing operators, and partner aggregators to embed State Farm renters insurance into tenant onboarding flows. Coverage is underwritten by the appropriate State Farm subsidiary for each state jurisdiction.

- **Documentation:** [https://developer.statefarm.com/api/renters](https://developer.statefarm.com/api/renters)
- **OpenAPI:** [openapi/state-farm-insurance-cos-renters-openapi.yml](openapi/state-farm-insurance-cos-renters-openapi.yml)

### B2B Lender Services API

Provides mortgage lenders and auto lenders with programmatic access to verify insurance coverage for collateral assets financed by their borrowers across any State Farm Insurance Companies subsidiary.

- **Documentation:** [https://b2b.statefarm.com/b2b-content/home-auto-lenders/ins-inquiry](https://b2b.statefarm.com/b2b-content/home-auto-lenders/ins-inquiry)
